# Amazon_Prime_Video_Content_Analytics_Dashboard_Power_BI
Built an interactive Power BI dashboard to analyze Amazon Prime Video content data and uncover insights related to content distribution, ratings, genres, and release trends.

	Key Points:
•	Created KPIs for total titles, genres, directors, ratings, and release year range.
•	Analyzed genre-wise and rating-wise content distribution.
•	Visualized country-wise content availability using map charts.
•	Compared Movies vs TV Shows and tracked year-wise release trends.
•	Enabled interactive analysis using slicers and filters.
•	Tools: Power BI, DAX, Data Modeling, Data Visualization
